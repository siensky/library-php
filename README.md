# PHP Library API (Practice Project)

## Overview

This is a simple PHP project built to practice backend development without using any frameworks.
It is a minimal version of a previous JavaScript library API.

The app allows you to view books and authors, see individual items, and borrow books.

## Features

* View all books
* View all authors
* View single book
* View single author
* Borrow a book
* Simple UI with CSS
* Built with plain PHP (no frameworks)
* Docker support for deployment

## Tech Stack

* PHP
* PostgreSQL
* Docker
* CSS


## Purpose

This project was built to practice:

* PHP fundamentals
* Working without frameworks
* Basic API structure
* Using PostgreSQL
* Deploying with Docker

## Notes

This is a learning project and not meant for production.
